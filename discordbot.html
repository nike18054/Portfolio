Discord Bot

with node.js & discord.js

Requirements

  1. Node.js
  2. Discord.js
  3. A Discord account

To download discord.js type this in your command line

npm install --save discord.js

Setting up your Discord Bot

  1. Create a project folder and put discord.js in it
  2. Download/Clone this rep (put it in your project folder)
  3. Create your bot here https://discordapp.com/developers/applications/me
  4. Invite bot to Discord Server at this Link https://discordapi.com/permissions.html (requires client ID bottom left)
    5. MAKE SURE YOU GIVE THE PERMISSIONS IT NEEDS IN ORDER TO CARRY OUT YOUR CODE CORRECTLY
  5. Open index.js and enter the User Token of your bot in the bottom(this allows the bot to be accessed by this code)
  6. On the command line navigate to your project folder. ex "cd C:\Windows\Users\me\myprojectfolder" once in the project directory type node . (this makes the current code live)

Examples

bot.on('message', (message) =>{
   if (message.content == '!test'){
      message.author.send('Whatever message you would like in reply\n\n'+
         'The formating is up to you.'
         );
   }
});

Creating a command to reply to

When someone types a command in the chat your bot should reply. The command specified can be set with

if (message.content == '!test')

with the !test being the command typed in the chat. You can change this to any command you would like but remember it is case sensitive and spelling has to be exact in order for the bot to reply. There is a way to set an alias command to output the same message but it is much more complex and I am only showing you the beginners for making a functioning Discord bot
